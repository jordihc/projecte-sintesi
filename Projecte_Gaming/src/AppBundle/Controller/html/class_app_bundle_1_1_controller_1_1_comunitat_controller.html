Detailed Description

Aquesta classe,ComunitatController s'utilitza per a enmagatzemar i controlar totes les funcions i accions que tenen relació amb la comunitat.

Aquesta classe hereda elements de la classe controller que és la classe què contè els elements fonamentals de tots els *conroladors.
